# Red Liberty

Thank you for downloading Red Liberty! Please be aware, this is the in-dev version of the mod, and as such may be unstable.

**Installation**
1) Download the mod files and place them in a folder of your choice.
2) Open the HoI4 launcher, navigate to "Upload Mod", "Create a Mod" and create a blank mod using the menu.
3) Copy and paste the entire contents of the github repo into the blank mod file you just created in "...\Documents\Paradox Interactive\Hearts of Iron IV\your_mod_name".
4) Enable the mod in the launcher and enjoy.
